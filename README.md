# Aquarium Water Change Reminder

This program reminds aquarium owners when it is time to perform a water change.  
The user enters the last water change date and the program calculates when the next one should happen.

This type of tool could help aquarium hobbyists maintain healthier tanks by preventing missed maintenance.

---

## Features
- Enter last water change date
- Calculates next recommended water change
- Displays reminder to the user

This project uses standard React Native and JavaScript libraries.
